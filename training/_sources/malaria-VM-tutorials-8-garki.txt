===============================================================
Vector and malaria 8: Garki project retrospective - IRS and MDA
===============================================================


This tutorial examines the historical combination of :term:`indoor residual spraying (IRS)` and
:term:`mass drug administration (MDA)` in the Garki District of Nigeria. This intervention contains
two events: an **IRSHousingModification** event and an **AntimalarialDrug** event.


Use the templates (config.json and campaign.json) in the following directory for this tutorial.

* Scenarios\VectorAndMalaria\08_Garki_IRSMDA



Steps
=====

#.  Set **Demographics_Filename** to “Garki_single_demographics.compiled.json”. For
    more information, see :ref:`config-inputfiles`.

#.  Set the disease parameters for “MALARIA_SIM” as noted in :doc:`malaria-VM-tutorials-2-full`.

#.  Configure the **IRSHousingModification** event. For more information, see :ref:`iv-irshousingmod`
    parameters.

    * Set **Start_Day** to 2190.
    * Set **Demographic_Coverage** to 0.8.
    * Set **Number_Repetitions** to 4.
    * Set **Secondary_Decay_Time_Constant** to 3650.
    * Set **Timesteps_Between_Repettions** to 365.

#.  Configure the **AntimalarialDrug** event. For more information, see :ref:`iv-antimalarialdrug`
    parameters.

    * Set the **Start_Day** to 2190.
    * Set **Cost_To_Customer** to 3.75.
    * Set **Demograpchi_coverage** to 0.8.
    * Set **Dosing_Type** to "FullTreatmentParasiteDetect."
    * Set **Drug_Type** to "Chloroquine."
    * Set **Number_Repetitions** to 9.
    * Set **Timesteps_Between_Repetitions** to 60.

Run the simulation
==================

Run the simulation and generate graphs of the simulation output. For more information, see
:doc:`using-quickstart`.  Compare to baseline and bednets alone after running the simulation.
